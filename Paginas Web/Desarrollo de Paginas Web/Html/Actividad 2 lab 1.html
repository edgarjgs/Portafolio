Recomendación de Hardware y Software para el Sistema de Servicio de Declaraciones y Pagos

Las características de Hardware que aparecen en la siguiente tabla permitirán utilizar el Sistema de Servicio de Declaraciones y Pagos (Pago Referenciado). Sin embargo, se recomienda tener entre 2 GB y 4 GB de RAM, y un monitor con resolución de 1024 x 768 para un rendimiento óptimo. Esto tomando como base que un usuario puede abrir varias aplicaciones ocupando mayor cantidad de memoria cada una; el resto será utilizado por los procesos del sistema operativo; y la velocidad del procesador afecta el renderizado (generación binaria a partir de números reales) de las páginas HTML.

Características mínimas de los equipos de Cómputo

Componente Descripción
Procesador Intel i5 2.8 Ghz (mínimo recomendado)
Memoria RAM 2 GB RAM
Disco duro 500 GB de espacio
Periféricos Mouse y teclado en español
Puertos de entrada/salida 2 puerto usb
Comunicación por cada PC Tarjeta de red Ethernet, considerando un ancho de banda para la conexión a internet de 100 Mb.
Monitor Monitor a color con resolución mínima de 1024 x 768 pixels.
Impresora La impresora deberá estar conectada en red.

Software requerido para el funcionamiento del Sistema de Servicio de Declaraciones y Pagos

Componentes Instalación/Configuración
Java Virtual Machine Instalar o actualizar Java Virtual Machine (JVM), para poder ejecutar las aplicaciones Java existentes en la aplicación del Servicio de Declaraciones y Pagos, tal como es el Formulario Electrónico.
Acrobat Reader Software gratuito Adobe Reader, el cual permitirá ver e imprimir archivos PDF (Portable Document Format, por sus siglas en inglés).
Silverlight versión 4.0 Software gratuito versión 4.0
